# Pandas CSV Reader & Analysis

## Description
This project demonstrates how to read and analyze data using Pandas.

## Features
- Read Excel file
- Data exploration (head, tail, info)
- Statistical analysis (mean, median, min, max)
- Filtering data
- Sorting values
- Saving processed data

## Tools Used
- Python
- Pandas

## Outcome
Learned how to handle and analyze datasets efficiently using Pandas.
